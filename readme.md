（三）作品文档

主题：

作品内容包括影视作品《fate》、《鬼灭之刃》、《崩坏 3rdCG 女王降临》、《咒术回战》等，作品主题是战斗、MAD；目的是让观众体会战斗的热血和战斗前后的平静，也让制作者体验剪辑 MAD 的过程。

设计与实现：

按照bgm《Whatever It Takes》的各个踩点用pr填充视频、音频素材和视频剪辑效果，用pr、ae和au制作一定的视频、音频效果。
